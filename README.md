# ATS技术周报

通过周报的形式把一些质量较高的原创，翻译和转载的文章收集起来，分享给团队，提高团队的技术水平。

## 内容

- 原创文章
- 翻译文章
- 转载文章
- 开源项目

需要附上原文链接和至少一位编辑的点评内容，点评不少于20字。

## 形式

以`markdown`格式在`gitlab`上进行协作，每周生成一封邮件发送给ATS一部的全体成员

## 目标

建设成类似于[tw技术雷达](https://www.thoughtworks.com/cn/radar)一样高水准，有影响力的技术刊物
